# n-back

http://ccd-school.de/coding-dojo/application-katas/n-back


## Prerequisites

* node v9.5.0
* npm v5.6.0
* yarn v0.23.4

```
brew install node@9
npm install -g npm@5.6.0
npm install -g yarn@0.23.4
yarn install // install node_modules
```

## execute program
```
node n-back.js
```

## run tests

```
node n-back.js --test
node n-back.js --test | npx faucet // pretty print TAP output

```
